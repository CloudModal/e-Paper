# 欢迎浏览此项目，如果喜欢请 Star

**e-Paper 是个电子日历项目**


![Preview](/Preview.jpg "Preview")


#### 硬件信息 (默认)
- 树莓派 zero w/3/3B/4
- 电子墨水屏（7.5寸）
- DHT22 （选填）

#### 软件
- python3

## 系统安装

*默认你已经安装好了系统，此步骤省略，如果不会请看*[这里](https://zhuanlan.zhihu.com/p/59027897 "这里")

准备就绪后，克隆本项目

执行命令安装依赖：`pip3 install -r requirements.txt`

最后执行 `python3 e-paper-render.py`
